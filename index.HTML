How to run

  1. Save this file as bus_ticket_booking.php in your server folder (htdocs/www).
  2. Make sure the folder is writable so bookings.csv can be created (chmod 755/775 or as needed).
  3. Open it in browser via http://localhost/bus_ticket_booking.php.
Note: This demo uses a CSV file for storage — for production, use a database + stronger validation and authentication.

Security & Production Notes

  • Sanitize and validate inputs more strictly.
  • Use prepared statements and a database (MySQL/Postgres) instead of CSV for concurrency.
  • Protect against CSRF and add user authentication.
